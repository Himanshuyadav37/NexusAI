## вҡЎ What is NeuroForge?
```
> **NeuroForge AI** is not just another AI chatbot вҖ” it's a full **Autonomous Multi-Agent Operating System** that treats LLMs as dynamic processing cores.
>
> Send it an idea. It **Plans вҶ’ Codes вҶ’ Tests вҶ’ Debugs вҶ’ Deploys** вҖ” completely on its own.
```

## рҹҢі Engineer AI вҖ” Autonomous Pipeline (Tree View)
```
`
рҹ’Ў Your Idea
в”Ӯ
в””в”Җв”Җв–ә рҹ§  PLANNER AGENT
     в”Ӯ   Reads your request вҶ’ builds architectural blueprint
     в”Ӯ   Queries ChromaDB for relevant code templates
     в”Ӯ   Outputs: file list, dependencies, architecture spec
     в”Ӯ
     в””в”Җв”Җв–ә рҹ’» CODER AGENT
          в”Ӯ   Iterates over every file in the plan
          в”Ӯ   Writes clean, modular, production-ready code
          в”Ӯ   Injects RAG context from ChromaDB memory
          в”Ӯ
          в””в”Җв”Җв–ә рҹ§Ә TESTER AGENT
               в”Ӯ   Runs AST parsing + lint checks on all files
               в”Ӯ   Verifies imports, syntax, and logic
               в”Ӯ   Outputs structured bug report per file
               в”Ӯ
               в”ңв”Җв”Җв–ә [вқҢ CRITICAL BUGS FOUND]
               в”Ӯ         в”Ӯ
               в”Ӯ         в””в”Җв”Җв–ә рҹ”§ DEBUGGER AGENT
               в”Ӯ                   Traces each bug to source line
               в”Ӯ                   Applies targeted fix (no side effects)
               в”Ӯ                   Increments iteration counter
               в”Ӯ                   в”Ӯ
               в”Ӯ                   в””в”Җв”Җв–ә рҹ”Ғ Back to TESTER AGENT
               в”Ӯ                        (loop until all tests pass
               в”Ӯ                         or max iterations reached)
               в”Ӯ
               в””в”Җв”Җв–ә [вң… ALL TESTS PASS]
                         в”Ӯ
                         в””в”Җв”Җв–ә рҹҡҖ DEPLOYER AGENT
                                   Generates Dockerfile + Compose
                                   Builds Kubernetes pod specs
                                   Drafts cloud provider setup guide
                                   в”Ӯ
                                   в””в”Җв”Җв–ә рҹ’ҫ Saved to MongoDB
                                             в”Ӯ
                                             в””в”Җв”Җв–ә рҹ“Ў SSE Stream
                                                       в”Ӯ
                                                       в””в”Җв”Җв–ә рҹ–ҘпёҸ Your Browser
`
